Question 8. Whatâ€™s the difference between window vs document?

Window

1. Window object represents the browser window or the Global javascript environment in which the web page is running. 
2. Window object provides properties and methods for interacting with the browser window itself. 
     Such as opening new windows or tabs, resizing the window, accessing the URL, and handling events like scrolling or resizing.
3. It also serves as the global object for JavaScript, meaning that variables and functions defined globally are attached to the window object.
4. For example, window.alert() is used to display an alert dialog.

Document

1. The document object represents the HTML document or the web page loaded in the browser window.
2. It provides access to the elements and content within the document, allowing you to manipulate or modify them dynamically.
3. The document object offers methods to select and modify elements, create new elements, modify styles and classes, handle events, and more.
4. For example, document.getElementById() is used to retrieve an element by its ID, and document.createElement() is used to create a new HTML element.
